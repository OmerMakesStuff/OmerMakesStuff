# Banner generator

This is a script to generate a banner image for this profile repo. It grabs the [Omer Makes Stuff logo](https://github.com/OmerMakesStuff/logo/blob/main/omer-makes-stuff/omer-makes-stuff.svg) and puts it on a 2560×640 banner image, then saves that to `banner.png` at the root of the repo.

## Using the script

Using Node.js & PNPM:

```sh
pnpm i && pnpm start
```
